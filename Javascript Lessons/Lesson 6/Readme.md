# Objects in JavaScript

Objects store data in **key-value pairs**.

## Example

```js
let person = {
  name: "John",
  age: 25,
  country: "Azerbaijan"
};

console.log(person.name); // John
console.log(person["age"]); // 25

Adding new properties
person.job = "Developer";
console.log(person.job); // Developer
