# Functions in JavaScript

Functions are reusable blocks of code.

## Example

```js
function greet(name) {
  return "Hello, " + name;
}

console.log(greet("Alice")); // Hello, Alice

// Arrow Functions
const add = (a, b) => a + b;
console.log(add(5, 3)); // 8
