# Loops in JavaScript

Loops allow you to repeat a block of code multiple times.

## for loop

```js
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);
