# Operators in JavaScript

Operators are special symbols used to perform operations on values and variables.

## Types of Operators
- Arithmetic (`+`, `-`, `*`, `/`, `%`)
- Assignment (`=`, `+=`, `-=`, `*=`)
- Comparison (`==`, `===`, `!=`, `>`, `<`)
- Logical (`&&`, `||`, `!`)

## Example

```js
let a = 10;
let b = 5;

console.log(a + b); // 15
console.log(a > b); // true
console.log(a == "10"); // true
console.log(a === "10"); // false
